# Prompt Suggestions

_Draft — pending operator approval._

Copy-paste starters for driving Do It List through the `doitlist` MCP and its skill. Deliberately minimal: each names the source and the destination, nothing else. Scope, numbering, grain, and questions are the skill's job — a prompt that coaches them hides skill gaps.

## Import an existing plan

> Import this repo's plan into a new DoItList Initiative.

The agent locates the plan (a `PLAN.md` or equivalent), follows its references, and sorts plan content from reference-only docs — expect a scope question where that split isn't obvious.

## Include in a one-shot app build

Append to a build-me-an-app-in-one-prompt request:

> Track the build plan as a DoItList Initiative and keep it current as you build.

## Plan mode

When an agent is about to draft an implementation plan:

> Put the plan in a DoItList Initiative instead of a doc, and execute from it.

## Work an existing Initiative

> Work from the ⟨name⟩ Initiative in DoItList; keep progress and the journal current as you go.
