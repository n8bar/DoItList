# Prompt Suggestions

_Draft — pending operator approval._

Copy-paste starters for the `doitlist` MCP + skill. The skill carries the conventions; a prompt is verb + source + destination. Anything more is coaching, and coaching hides skill gaps.

## Import an existing plan

> Import PLAN into doitlist.

The agent finds the plan doc, follows its references, sorts plan from reference — and asks when the split isn't obvious.

## Include in a one-shot app build

Append to the build prompt:

> Track the plan in doitlist as you build.

## Plan mode

> Plan into doitlist; execute from it.

## Work an existing Initiative

> Work from ⟨name⟩ in doitlist.
